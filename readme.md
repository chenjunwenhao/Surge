一个干净利落的 MySQL 管理工具。不做 PostgreSQL，不做 SQLite，只做 MySQL 一件小事。

### 三个核心

**搜索快。** 在侧边栏输入任何字符，即时过滤数据库、表、字段，不用一层层点开找。

**格式化好。** 一键格式化 SQL，按人类阅读习惯排版，不是机器式的关键字换行。

**直通数据。** 双击表名直接看数据，展开即见列结构，无需手写 `SELECT *`。

### 快速开始

```bash
git clone git@github.com:chenjunwenhao/MySQL-Explorer.git
cd MySQL-Explorer
npm install
npm run dev
```

打开 `http://localhost:5173`，填上连接信息就行。

### 更多能力

暗/亮双主题 · Monaco 编辑器 · 双击编辑单元格 · DDL 查看 · 索引查看 · 导出 CSV/JSON · 查询历史 · AES-256 加密 · Electron 桌面版
